💡 TIPS ÚTILES:

  1. Primer intento falla pero luego funciona:
     → Normal. El sistema reintentar automáticamente.
     → Espera 2-3 segundos y verás el navegador.

  2. Múltiples cuentas de Microsoft:
     → Usa navegador en modo incógnito para limpiar cookies.
     → Así evitas iniciar sesión con la cuenta equivocada.

  3. Cuentas de trabajo no funcionan:
     → Microsoft bloquea consentimiento para cuentas corporativas.
     → Usa cuenta personal: Outlook.com, Hotmail.com, Live.com

  4. Puerto 3000 ocupado:
     → El launcher intentará automáticamente con puerto 8080.
     → Cierra aplicaciones que usen puerto 3000 si es necesario.

  5. Sesión lenta o timeout:
     → Asegúrate de tener conexión a internet estable.
     → Completa el login dentro del tiempo límite (5 minutos).
